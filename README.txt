This is a chrome extension found at 
https://chrome.google.com/webstore/detail/ukulele-tab-player/ggmbngnjkkcghbmbpnclfbcihakjidoc

Ukulele tab reader reads four lines of selected ukulele tablatured text and plays it within the chrome browser.
